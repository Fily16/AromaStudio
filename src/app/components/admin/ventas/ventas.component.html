Como funciona:

  1. Configura la URL del backend arriba y guarda
  2. Haz click en "Sincronizar Stock" para enviar productos al Sheet
  3. Registra ventas en el formulario - el stock se descuenta en el Sheet
  4. Al sincronizar de nuevo, las ventas del Sheet se importan al backend automaticamente
  5. El catalogo publico se actualiza con el stock correcto

Instrucciones de configuracion

  1. Crea un Google Sheet nuevo
  2. Ve a Extensiones > Apps Script
  3. Pega el codigo del archivo google-sheets/apps-script.gs
  4. Ejecuta la funcion setup (boton Play)
  5. Acepta los permisos de Google
  6. En el registro de ejecucion aparece la URL del formulario - copiala
  7. Ve a Implementar > Nueva implementacion > App web
  8. Pon "Ejecutar como: Yo" y "Acceso: Cualquier persona"
  9. Copia la URL del Web App y pegala arriba
  10. Guarda la URL del formulario en tu celular para acceso rapido
