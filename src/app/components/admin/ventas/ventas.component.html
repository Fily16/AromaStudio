Como configurar Google Sheets

  1. Crea un Google Sheet nuevo
  2. Ve a Extensiones > Apps Script
  3. Pega el codigo del archivo google-sheets/apps-script.gs
  4. Ejecuta la funcion setup (boton Play)
  5. Acepta los permisos de Google
  6. Ve a Implementar > Nueva implementacion > App web
  7. Pon "Ejecutar como: Yo" y "Acceso: Cualquier persona"
  8. Copia la URL del Web App
  9. Ve a Config y agrega estas claves:
    • google_script_url = URL del Web App
    • google_form_url = URL del formulario (lo da el setup)
    • google_sheet_url = URL del Google Sheet
  10. Guarda la URL del formulario en tu celular para acceso rapido
